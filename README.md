1. Принцип сложения – если большая цифра стоит перед меньшей, то

2. Принцип вычитания – если меньшая цифра стоит перед большей,
то меньшая вычитается (например, IV = 5 - 1 = 4).

число.

Пример 2: ВВОД: “LXXXI”, ВЫВОД: 81
